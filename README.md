# Smart Snake - Reinforcement Learning Snake Game

A classic Snake game powered by Deep Q-Network (DQN) reinforcement learning.
The snake learns to play by itself through trial and error, gradually improving
its score over hundreds of training episodes.

```
  +------------------------------------------+
  |                                          |
  |   [##][##]                               |
  |        [##][##]    [##][##] (o)          |
  |            [##][##][##]                  |
  |                                          |
  +------------------------------------------+
    Score: 8
```

## How It Works

### The State (11 features)

The agent sees the world as 11 numbers:

```
[danger_straight, danger_right, danger_left,
 dir_up, dir_down, dir_left, dir_right,
 apple_up, apple_down, apple_left, apple_right]
```

- **Danger (3):** Is there a wall or self-collision straight ahead, to the right, or to the left?
- **Direction (4):** Which way is the snake currently moving? (one-hot)
- **Apple (4):** Which direction is the apple relative to the snake's head? (one-hot)

### The Action (4 choices)

```
0 = UP
1 = DOWN
2 = LEFT
3 = RIGHT
```

### The Reward

```
+10   Eating an apple
-10   Hitting a wall or yourself
  0   Any other move
```

### The Neural Network

A 3-layer feedforward network:

```
Input(11) -> Linear(256) -> ReLU -> Linear(256) -> ReLU -> Linear(4)
```

- **Input:** 11 state features
- **Output:** 4 Q-values (one per action)
- The agent picks the action with the highest Q-value.

### The DQN Algorithm

1. The agent observes the current state (11 features).
2. It picks an action using epsilon-greedy strategy:
   - With probability epsilon: random action (exploration)
   - Otherwise: best action according to the network (exploitation)
3. The environment returns the next state, reward, and whether the game is over.
4. The experience (state, action, reward, next_state, done) is stored in a replay buffer.
5. Every step, the agent samples a random batch from the buffer and trains the network using the Bellman equation:

```
target = reward + gamma * max(Q(next_state)) * (1 - done)
loss = MSE(predicted_Q, target)
```

6. Epsilon decays gradually from 1.0 to 0.01 so the snake transitions from random exploration to using its learned knowledge.

## Usage

### Train the Agent

Run the training loop. The snake will play 1000 games and save the model:

```
python train.py
```

You will see output like:

```
0: score-> 0 | total rewards-> 0 | epsilon-> 0.995
1: score-> 0 | total rewards-> 0 | epsilon-> 0.990025
...
500: score-> 3 | total rewards-> 150 | epsilon-> 0.08
...
1000: score-> 8 | total rewards-> 420 | epsilon-> 0.01
```

After training completes, `training_test.pth` will be saved with the learned weights.

### Watch the Trained Snake Play

After training, run the play script to watch 20 games:

```
python play.py
```

The script uses epsilon=0 (pure exploitation) so the snake plays entirely
using what it learned. Output:

```
episode: 0 | score: 5
episode: 1 | score: 3
...
BEST SCORE: 8
```

### Adjust Training Speed

To watch the training in real-time (slower), you can uncomment the
`time.sleep(0.1)` line in `train.py`. By default it runs as fast as possible.

## File Details

### snake_game.py

The original game with `Snake` and `Apple` classes. Handles movement,
collision, rendering, and scoring. Not used directly during RL training.

### snake_env.py

Wraps the game into an RL environment with a Gym-like interface:

- `reset()` -- Starts a new game and returns the initial state.
- `step(action)` -- Takes an action, returns (next_state, reward, done).
- `get_state()` -- Returns the 11-feature state vector.
- `is_over()` -- Returns whether the game has ended.

### dqn_agent.py

The learning agent with three components:

1. **Neural Network:** 3-layer feedforward net (11 -> 256 -> 256 -> 4)
2. **Replay Buffer:** Stores up to 10,000 past experiences for batch training
3. **DQN Agent:** Epsilon-greedy action selection, training with the Bellman equation, and save/load functionality

Key methods:

```
store(state, action, reward, next_state, done)  -- Save an experience
sample(batch_size)                               -- Random batch for training
choose_action(state, epsilon)                    -- Pick an action
train(batch_size, gamma)                         -- Learn from past experiences
save(path)                                       -- Save model weights
load(path)                                       -- Load trained weights
```

### train.py

The training loop. Runs 1000 episodes, decaying epsilon from 1.0 to 0.01.
Each episode: reset -> play until game over -> store experiences -> train ->
decay epsilon. Saves the model when done.

### play.py

Loads a trained model and runs 20 games with epsilon=0 (no exploration).
Prints each game's score and the overall best score.

## Tips for Better Training

- **More episodes:** The snake needs ~500+ episodes to start scoring consistently.
  Increase `episodes` in `train.py` for better results.
- **Slower decay:** Lower `epsilon_decay` (e.g., 0.998) keeps the snake exploring longer.
- **Faster learning:** Increase `batch_size` (e.g., 128) for more stable training.
- **Long-term thinking:** Increase `gamma` (e.g., 0.95) to make the snake value future rewards more.
